CSS: CSS (Cascading Style Sheets)

É uma linguagem de estilo usada para descrever a apresentação de documentos HTML. Ela define como elementos devem ser exibidos na tela, em papel ou em outros meios.

Importância no Desenvolvimento Web:

Com CSS, podemos separar o conteúdo (HTML) da apresentação (estilos), o que facilita a manutenção e a leitura do código. CSS permite criar layouts responsivos, aplicar cores, fontes, espaçamentos, animações e muito mais.

Sintaxe html Sintaxe CSS
< h1>Título< h1> h1{ color: 'red'}
Tag de abertura + conteúdo + Tag de fechamento = Elemento Seletor = Elemento ou class(.container_css) ou Id(#menu)
