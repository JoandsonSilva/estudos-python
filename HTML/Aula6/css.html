    • CSS (Cascading Style Sheets)

    É uma linguagem de estilo usada para descrever a apresentação de documentos HTML

    Importância do Desenvolvimento Web

    Com CSS, podemos separar o conteúdo (HTML) da apresentação(estilos), o que facilita a manutenção e a leitura do código.

    Sintaxe html Sintaxe CSS
    Tag de abertura + conteúdo + Tag de fechamento = Elemento h1{ color: 'red'}
    < h1> Conteúdo < /h1> Seletor = Elemento ou class ou Id
